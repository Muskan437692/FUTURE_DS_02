# Customer Retention & Churn Analysis

## Project Overview
This project analyzes customer retention and churn behavior for a subscription-based business. The goal is to identify churn patterns, key retention drivers, and customer lifetime trends to support data-driven decision-making.

## Objectives
- Identify churn patterns and high-risk customer segments  
- Perform cohort and retention analysis  
- Analyze customer lifetime value trends  
- Provide actionable recommendations to reduce churn  

## Tools & Technologies
- Python (Pandas, Matplotlib, Seaborn)  
- Excel / Power BI  
- Jupyter Notebook  

## Key Analysis Performed
- Churn rate calculation  
- Cohort-based retention analysis  
- Customer lifetime metrics  
- Trend and behavior analysis  

## Deliverables
- Data cleaning and analysis notebooks  
- Retention and churn dashboards  
- Business insights with actionable recommendations  

## Key Insights
- Higher churn observed in early customer lifecycle  
- Long-term customers show higher lifetime value  
- Retention improves with consistent engagement  

## How to Run
Clone the repository and run the Jupyter Notebook to reproduce the analysis.
